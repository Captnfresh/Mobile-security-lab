# JADX-GUI

## What is JADX-GUI?
JADX-GUI is a decompiler that converts Android APK files back into readable Java code.

## Purpose in Mobile Security
- Reverse engineer compiled Android apps
- Analyze app source code for vulnerabilities
- Examine app permissions and components
- Find hardcoded credentials and API keys

## Key Features
- Decompiles APK to Java source code
- Shows AndroidManifest.xml
- Displays resources (images, layouts, strings)
- Search functionality across entire codebase

## Common Use Cases
- Security research and vulnerability assessment
- Malware analysis
- Learning how apps work internally
- CTF challenges (like Injured Android)

## Basic Usage
1. Open `jadx-gui.bat`
2. Drag and drop any APK file
3. Browse the decompiled code
4. Look for security issues
