# Prerequisites

## System Requirements
- **OS**: Windows 11 (64-bit)
- **RAM**: Minimum 8GB (16GB recommended)
- **Storage**: 20GB free disk space
- **Administrator access** required

## Required Software
1. **Java JDK 11** - [Download here](https://www.oracle.com/java/technologies/javase-jdk11-downloads.html)
2. **Docker Desktop** - [Download here](https://www.docker.com/products/docker-desktop)
3. **Android Platform Tools (ADB)** - [Download here](https://developer.android.com/studio/releases/platform-tools)

## Account Requirements
- Oracle account (for Java JDK download)
- Docker Hub account (optional)

## Verification Checklist
Before proceeding, ensure you have:
- [ ] Windows 10/11 64-bit
- [ ] 8GB+ RAM available
- [ ] 20GB+ free disk space
- [ ] Administrator privileges
